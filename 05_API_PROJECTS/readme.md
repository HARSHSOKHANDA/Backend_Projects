# 05_API_PROJECTS

This folder contains multiple API-related projects demonstrating various concepts and implementations in Node.js and Express. Each subfolder is a standalone project with its own code and dependencies.

## Projects Overview

### 01_JSON_API

- Simple RESTful API using Express.
- Demonstrates basic CRUD operations.
- Uses JSON data for recipes.
- Includes EJS views and custom CSS.

### 02_Axios

- Example of using Axios for API requests.
- Shows how to fetch data from APIs in Node.js/Express.
- Includes EJS views and custom CSS.

### 03_Authentication

- Demonstrates basic authentication concepts.
- Shows login flow and protected routes.
- Includes EJS views.

### 04_REST APIs

- REST API implementation with Express.
- Demonstrates HTTP methods and status codes.
- Includes EJS views.

### 05_DIY API

- Custom API project (e.g., Jokes API).
- Includes Postman collection for testing.
- Shows how to design and document your own API.

### 06_Blog_API_Project

- Blog API with CRUD operations.
- Includes both `index.js` and `server.js` for different setups.
- EJS views for blog management and modification.
- Custom CSS for styling.

### 07_Secrets Project

- Secrets sharing API/project.
- Demonstrates authentication and authorization.
- Includes image assets and custom CSS.
- EJS views for UI.

## How to Run

1. Navigate to the desired project folder.
2. Install dependencies:
   ```powershell
   npm install
   ```
3. Start the server:
   ```powershell
   node index.js
   ```
   (Some projects may use `server.js` instead of `index.js`.)

## Requirements

- Node.js
- npm

## Folder Structure

- `public/` - Static assets (CSS, images)
- `views/` - EJS templates for rendering UI
- `index.js` / `server.js` - Main server files
- `package.json` - Project dependencies

## Notes

- Each project is independent; install dependencies separately for each.
- Refer to individual project folders for more details and documentation.

---

Feel free to explore each project to learn about different API concepts, authentication methods, and Express.js features.
